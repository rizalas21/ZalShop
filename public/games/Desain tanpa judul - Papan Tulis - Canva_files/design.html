Kanvas adalah tempat desain Anda berada. Untuk menavigasi ke halaman, tekan Enter. Untuk menavigasi ke grup atau elemen di halaman, tekan Enter lagi. Tekan tab untuk pindah ke halaman, grup, atau elemen berikutnya. Untuk mengoperasikan bilah alat, gunakan tombol "Ctrl+F1". Jika elemen dipilih, pindahkan menggunakan tombol tanda panah; putar menggunakan tombol "," dan "."; dan ubah ukuran menggunakan tombol tanda panah bersama "Ctrl". Semua tombol ini dapat dimodifikasi dengan tombol "Shift" untuk menambahkan amplifikasi.
